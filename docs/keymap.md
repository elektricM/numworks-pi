# NumWorks Pi Keyboard Map

Power button toggles between **Keyboard** and **Mouse** mode.
`xnt` key activates Mode 0. `var` key activates Mode 1.
Shift (calculator Shift key) modifies keys — shown as `key / Shift+key` where applicable.

## Mouse Mode (toggle with Power)

Arrow keys become mouse movement. OK/Back become clicks.
All other keys behave the same as the active keyboard mode (xnt or var).

```
 ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐
 │  ←   │ │  ↑   │ │  ↓   │ │  →   │ │  OK  │ │ Back │
 │MOUSE←│ │MOUSE↑│ │MOUSE↓│ │MOUSE→│ │L-CLK │ │R-CLK │
 └──────┘ └──────┘ └──────┘ └──────┘ └──────┘ └──────┘

  Arrows move cursor continuously at ~50 Hz while held.
  Speed: 2 px per tick.
```

## Quick Reference — Special Characters

| Character | How to type |
|-----------|-------------|
| `.` | `.` key, mode 0 |
| `/` | `0` key, mode 0 |
| `?` | Shift + `0` key, mode 0 |
| `_` | Shift + `-` key, mode 1 |
| `[` | `(` key, mode 1 |
| `]` | `)` key, mode 1 |
| `{` | Shift + `(` key, mode 1 |
| `}` | Shift + `)` key, mode 1 |
| `;` | `.` key, mode 1 |
| `=` | EXE key, mode 1 |
| `*` | `×` key, mode 1 |
| `TAB` | Toolbox key (any mode) |
| `ESC` | Home key (any mode) or Backspace in mode 1 |
| `-` | `-` key, mode 1 |
| `+` | `+` key, mode 1 |
| `!` | Shift + `1` key, mode 1 |
| `@` | Shift + `2` key, mode 1 |
| `#` | Shift + `3` key, mode 1 |
| `$` | Shift + `4` key, mode 1 |
| `%` | Shift + `5` key, mode 1 |
| `^` | Shift + `6` key, mode 1 |
| `&` | Shift + `7` key, mode 1 |
| `(` | Shift + `9` key, mode 1 |
| `)` | Shift + `0` key, mode 1 |

## Still Missing

`\`, `~`, `` ` ``, `"`, `'`, `<`, `>`, `|`

These require remapping additional keys or adding a third keymap mode.

## Changes from Original (zardam) Keymap

| Key | Original | New | Why |
|-----|----------|-----|-----|
| Home | unmapped | ESC | ESC accessible in both modes |
| Toolbox | RALT | TAB | Tab completion in terminal |
| `0` key, mode 0 | KEY_QUESTION | KEY_SLASH | `/` for paths, Shift = `?` |
| `(` key, mode 1 | KEY_5 (bug) | KEY_LEFTBRACE | `[`, Shift = `{` |
| `)` key, mode 1 | KEY_MINUS | KEY_RIGHTBRACE | `]`, Shift = `}` |
| `-` key, mode 1 | KEY_KPMINUS | KEY_MINUS | `-`, Shift = `_` |
| `.` key, mode 0 | KEY_COMMA | KEY_DOT | `.` (period) |
